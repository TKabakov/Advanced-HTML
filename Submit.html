Action

The action attribute of the form specifies to which web address the form must be sent.

URL address:https://meta.com/login

Absolute path:/login

the address will be https://meta.com/login

Relative path:login

the address will be https://meta.com/company-info/login

When action attribute is omitted the form will be send to the currant address.

Method

The method attribute specifies which HTTP method is used to submit the form: GET or POST.

When method is not specified the default method is GET.

GET method - the data in the form field is encoded in the URL.

POST method - the date in the form field is part of the HTTP request body.
