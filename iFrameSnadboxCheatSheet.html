iFrame embeds an HTML page into another page.

allow

    It specifies what permissions of features are available to the frame:
  • allow="fullscreen" the fullscreen mode can be activated
  • allow="geolocation" lets you access the user's location

A semicolon-separator should be used to specify morw than one feature:

name

height

The default value is 150 pixels.

width

The default value is 300 pixels.

referrerpolicy

    A referrer is the HTTP header that lets the page know who is loading it. The common values are:
  • no-referrer The referrer header will not be sent.
  • origin The refferer will be limited to the origin of the referring page.
  • strict-origin The origin of the document is sent as referrer only when using the same protocol security level(HTTPS to HTTPS)
