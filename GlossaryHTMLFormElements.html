select

Defines a drop-down list of option.

It has a couple of attributes:

  • form, the id of the form in which the drop-down list appears
  • name specifies the name of the control
  • multiple Boolean attribute, when specified, indicates if a user can select mutiple options out of the list
  • required idicates, if the user is required to select an option before submitting a form
  • size mentions the number of visible oprtions in a drop-down list

textarea

Defines a multi-line input field, tipically to allow user to input longer textual data.

The common attributes include:

  • cols defines the width of the text area, the default value is 20
  • form the form element the text area is associated with
  • maxlength when specified, limits the maximum number of characters that can be entered
  • minlength defines hte minimum number of charcters that the user should enter
  • readonly once set, the user cannot modify the contents
  • rows defines the number of visible text lines for the text area
